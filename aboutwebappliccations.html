Web Applications

The definition of application software is a computer program designed to carry out a specific task other than one relating to the operation of the computer itself. Depending on the activity for which it was designed, an application can manipulate text, numbers, audio, graphics, and a combination of these elements. Some application packages focus on a single task, such as word processing, and some, called integrated software, include several applications. Applications may be bundled with computers, but usually, the term refers to applications or is called "apps" on mobile devices such as phones.
